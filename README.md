# SmartDisruptions Website (V1)

A proof engine and system demonstration platform showcasing AI-built web applications.

## Purpose

- Showcase AI-built web applications
- Demonstrate a repeatable AI orchestration system
- Establish credibility through working proof

## Branching Strategy

| Branch      | Purpose                                     |
| ----------- | ------------------------------------------- |
| `main`      | Production — deployed to live site          |
| `dev`       | Integration — all features merge here first |
| `feature/*` | Feature branches — individual work items    |

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
